# 👤 Profile Card UI

Um projeto simples e responsivo de **Cartão de Crédito/Débito**, desenvolvido para praticar conceitos de **HTML5** e **CSS3**, com foco em design moderno e organização visual.

---

## 📸 Preview do Projeto

Interface contendo:

- Foto de perfil
- Nome
- Links para redes sociais
- Efeito de virar
- Estatísticas do perfil

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**

---

## 🎨 Características do Design

- Layout moderno e minimalista
- Ícones de redes sociais
- Bordas arredondadas
- Efeitos de sombra
- Estrutura responsiva

---

## 🚀 Objetivo do Projeto

Praticar habilidades de desenvolvimento front-end através da criação de um componente visual elegante e reutilizável.
